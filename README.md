## Caso Rode o Código via VsCode
Para compilar e rodar este código, você vai precisar de:
* Compilador **GCC** instalado, o usado no projeto foi o modelo MSYS2.

## Como compilar e rodar

1. Rode o comando abaixo para gerar o .exe 
gcc main.c tetris.c interface.c -o tetris

## Para Executar o .exe

1. Rode o comando abaixo
.\tetris.exe
